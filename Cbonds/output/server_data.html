Тестовое задание

Задания, описанные ниже не сложные, но каждое их них может иметь некоторые подводные камни, обойти которые поможет умение разбираться в документации и решать возникающие проблемы. Задания связаны между собой логически, но если какое-либо задание сделать по каким-либо причинам не удается, можно приступать к следующему, задав ему необходимые входные данные. Скрипты должны работать на PHP 8.1.

Задача #1

Используя curl (http://php.net/manual/en/book.curl.php), получить контент с данной страницы (http://cbonds.info/sandbox/some_source.php) и записать его в файл (server_data.html)

Задача #2

Используя регулярные выражения PCRE (http://php.net/manual/en/book.pcre.php), разобрать данные из таблицы (Some table) и сформировать из них xml-файл (emissions.xml) следующего формата: 

<?xml version="1.0" encoding="utf-8"?>
<items>
	<item>
		<property title="Эмитент"><![CDATA[Открытое акционерное общество "АМУРМЕТАЛЛ"]]></property>
		<property title="Эмитент (кратко)"><![CDATA[ТГК-2]]></property>
		<!-- ... для каждого свойства в таблице -->
	</item>
	<item>
		<property title="Эмитент"><![CDATA[Открытое акционерное общество "Нефтяная компания "ЛУКОЙЛ""]]></property>
		<property title="Эмитент (кратко)"><![CDATA[ЛУКОЙЛ]]></property>
		<!-- ... для каждого свойства в таблице -->
	</item>
	<!-- ... для каждой строки в таблице -->
<items>
Для формирования xml-файла должен использоваться DOM (http://php.net/manual/en/book.dom.php).

Задача #3

Предположим, что данные из таблицы "some table" требуется сохранять в БД: какой на ваш взгляд должна быть ее структура?
Результатом должен быть sql_dump БД, структура таблиц которой реализует возможность хранения данных в нормализованном виде.

Задача #4

Составить запрос, который бы агрегировал данные из таблиц задания #3 по столбцу "Тип", суммиря столбец "номинал". Записи неопределенного типа также должны войти в выборку. Сортировка должна вестись по убыванию суммы номиналов.
Итоговая выборка в запросе должна содержать поля:
Тип | Номинал | Список эмитентов данного типа (перечисленных череp запятую).
